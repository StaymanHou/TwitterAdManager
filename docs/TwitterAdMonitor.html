TwitterAdMonitor¶

class Lib.TwitterAdMonitor.TwitterAdMonitor¶

This class operates as a monitor. It keeps updating data between local database and twitter server. Call Lib.TwitterAdMonitor.TwitterAdMonitor.start() to start the monitor. The function the process performs is implemented in the static method Lib.TwitterAdMonitor.TwitterAdMonitor.OperateFunction().

static OperateFunction()¶

This is a static method. It setups a number of threads including Lib.TwitterAdForeman.TwitterAdForeman and Lib.TwitterAdWorker.TwitterAdWorker according to the config.ini. You can modify the value of ‘[Monitor] - worker_thread_max_num’. You must have at least one thread. After the setup, it starts all of the threads, and wait for the threads join.

return a boolean indicates if myprocess is alive

join()¶

Wait for the process to join. Call Lib.TwitterAdMonitor.TwitterAdMonitor.myprocess.join().

myprocess = None¶

hold a multiprocessing.Process.

start()¶

Start the process. Call Lib.TwitterAdMonitor.TwitterAdMonitor.myprocess.start().

terminate()¶

Terminate myprocess.
