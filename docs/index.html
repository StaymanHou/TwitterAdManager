Welcome to TwitterAdManager’s documentation!¶

This is the detailed documentation of TwitterAdManager. It’s under the alpha release, so it’s unstable. You can find all the modules, classes in this project in the content below. I’m still working on it to fix bugs. If you want, you can write to me.

1. TwitterAdManager¶

class TwitterAdManager.TwitterAdManager is the top_level class in this project. To run the tool, simply initialize a TwitterAdManager object. Call TwitterAdManager.TwitterAdManager.start() to start and TwitterAdManager.TwitterAdManager.join() to wait.

>>> TAM = TwitterAdManager()
>>> TAM.start()
>>> TAM.join()
