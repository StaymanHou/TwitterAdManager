DateTimeHelper¶

This is the helper for datetime.datetime. Dealing with the operations related to datetime.

Lib.DateTimeHelper.floorbyday(input_datetime, hour_shift=0)¶

Return the floored datetime versus day from input_datetime.

Lib.DateTimeHelper.floorbyhour(input_datetime)¶

Return the floored datetime versus hour from input_datetime.

Lib.DateTimeHelper.oneday = datetime.timedelta(1)¶

A one day timedelta object.

Lib.DateTimeHelper.onehour = datetime.timedelta(0, 3600)¶

A one hour timedelta object.

Lib.DateTimeHelper.oneminute = datetime.timedelta(0, 60)¶

A one minute timedelta object.

Lib.DateTimeHelper.tenminutes = datetime.timedelta(0, 600)¶

A ten minutes timedelta object.

Lib.DateTimeHelper.timetoclock(cal_time)¶

Return an int indicates the clock of the cal_time. The value range from 0 to 23.
