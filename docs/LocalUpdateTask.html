LocalUpdateTask¶

class Lib.LocalUpdateTask.LocalUpdateTask(twitter_session, hour_start)¶

Bases: Lib.TwitterMonitorTask.TwitterMonitorTask

This task will hold a reference of a Lib.TwitterSession.TwitterSession while initialized and assign a check hour. Perform as a task to update the local database from twitter server, including campaign data and summary data. Set non-online-exist campaigns to ‘dead’. Create non-local-but-online-exist campaigns and set to ‘alive’.

create_new()¶

Create non-local-but-online-exist campaigns and set to ‘alive’ by calling CampaignHelper.create().

do()¶

return negative if fail, return 0 if succeed Calls following self methods in order. If any one of them failed, return failed code immediately. Lib.LocalUpdateTask.LocalUpdateTask.get_local_list() > Lib.LocalUpdateTask.LocalUpdateTask.get_online_list() > Lib.LocalUpdateTask.LocalUpdateTask.update_summary() > Lib.LocalUpdateTask.LocalUpdateTask.remove_camp_local_no_online() > Lib.LocalUpdateTask.LocalUpdateTask.update_exist() > Lib.LocalUpdateTask.LocalUpdateTask.create_new() >

get_local_list()¶

Set self.camp_local_list with the alive local campaign list by calling Lib.TwitterCampaign.TwitterCampaign.get_list()

get_online_list()¶

Set self.camp_online_list with the alive online campaign list. Not only this hour’s list but also this hour’s data. Loop to fetch datas of each page. Max loop number is set in config.ini [‘Monitor’ - ‘max_campaign_scan_pages’]. If there’s no more pages the loop will break. If the id is less than the minimum id in database, the loop will break, too. The minimum if is found by calling Lib.CampaignHelper.find_min_id().

perform()¶

invoke Lib.LocalUpdateTask.LocalUpdateTask.do() until success. Success means the self.do() method returns 0 See abstract method Lib.TwitterMonitorTask.TwitterMonitorTask.perform().

remove_camp_local_no_online()¶

self.camp_local_list is modified so that there is no dead

update_exist()¶

Update the datas of local-exist campaigns based on self.camp_online_list. self.camp_online_list is modified so that there is no exist

update_summary()¶

Update the summary based on self.camp_online_list.
