TwitterAdManager¶

class TwitterAdManager.TwitterAdManager¶

This is the top class of this project. It holds two process: Lib.TwitterAdMonitor.TwitterAdMonitor and Lib.TwitterAdMonitor.TwitterAdMonitor. To start it call TwitterAdManager.TwitterAdManager.start(). Call TwitterAdManager.TwitterAdManager.join() to wait.

TAController = None¶

hold the Lib.TwitterAdController.TwitterAdController process.

TAMonitor = None¶

hold the Lib.TwitterAdMonitor.TwitterAdMonitor process.

join()¶

Wait for the two processes to join.

start()¶

Start the two processes: TwitterAdManager.TwitterAdManager.TAMonitor process and TwitterAdManager.TwitterAdManager.TAController process.

stop()¶

Stop the processess if running.(Not implemented yet.)
