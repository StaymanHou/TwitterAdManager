TwitterAdController¶

class Lib.TwitterAdController.TwitterAdController¶

This class operates as a controller. It controls the ads on the local database. It does not perform any communication with twitter server. It just checks if there are enough ads alive, generate new campaigns, and delete poorly-performing ads.

static OperateFunction()¶

This is a static method. Periodically, it checks all active accounts. If the account meets the condition of commit a new check, it will do the following instructions. Get poorly-performing campaign list by calling Lib.CampaignHelper.get_poor_performance_camp_list(). Put the poor campaigns into delete_pending by calling Lib.CampaignHelper.set_delete_pending(). If number of alive campaigns is less than max_campaign_num, generate create_pending campaigns to full by calling Lib.CampaignHelper.generate_createpending_camp().
