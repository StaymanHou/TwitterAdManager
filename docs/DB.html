DB¶

class Lib.DB.DB¶

DB object. Responsible for communicating with the database in the entire program. Usage:

>>> db = DB()
>>> cur = db.execute('SELECT * FROM Accounts WHERE 1')
>>> ...
execute(exetuple)¶

Return the cursor which returned after the execution of the exetuple

..note:

The exetuple should be something like this:
                
(('MySQL query here where %s = %s'),('id',1))
get_connect()¶

Return the db connect.
