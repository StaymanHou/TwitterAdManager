class Lib.CreateTask.CreateTask(twitter_session, camp)¶

Bases: Lib.TwitterMonitorTask.TwitterMonitorTask

This task will hold a reference of a Lib.TwitterSession.TwitterSession while initialized. Perform as a task to create a ‘create_pending task’ onto twitter server, and update the localstatus to ‘alive’(or ‘create_fail’ if failed).
