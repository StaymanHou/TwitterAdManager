TwitterAdWorker¶

class Lib.TwitterAdWorker.TwitterAdWorker(TaskQueue, TaskLock)¶

Bases: threading.Thread

Worker extends threading.Thread. After starting, it will periodically check the TaskQueue. If there are works fetchable, and fetch and work on the task. Upon finishing the task, it opens the corresponding tube of the TaskQueue.

TaskQueue = None¶

It is responsible for the communication between Lib.TwitterAdForeman.TwitterAdForeman thread and Lib.TwitterAdWorker.TwitterAdWorker threads. It is an object of Lib.MultiTubeQueue.MultiTubeQueue.

check()¶

Loop until no task to do. Within the loop, it checks the TaskQueue. If get a task, it calls Lib.Task.Task.perform() of the task. Upon finishing the task, it opens the corresponding tube so that no two works will work on the tasks of a same account simultaneously.

run()¶

Implement the interface of Thread. it Periodically calls Lib.TwitterAdWorker.TwitterAdWorker.check().
