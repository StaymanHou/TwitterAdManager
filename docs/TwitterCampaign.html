TwitterCampaign¶

class Lib.TwitterCampaign.TwitterCampaign¶
A Twitter Campaign object. Contains id, name,
status, data and etc. It uses Lib.DB.DB.

..note:

self.data is stored as a JSON object in database
static get_alive_createpending_num(fi_id)¶

Return the number of the Campaigns of the account which are alive or create_pending.

static get_list(fi_id, local_status=None)¶

Return a list of Lib.TwitterCampaign.TwitterCampaign of a give fi_id. local_status can be specified.

save()¶

Save the campaign to database. If it has non-zeor pk, it will do UPDATE. Otherwise, do INSERT.
