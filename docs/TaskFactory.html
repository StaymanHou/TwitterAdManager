TaskFactory¶

class Lib.TaskFactory.TaskFactory(twitter_session)¶

TaskFactory is a Factory for generating Lib.Task.Task`s. A factory must be initialized with a :class:`Lib.TwitterSession.TwitterSession. Every Task it generates will be given a twitter_session reference which was setted upon initialization.

get_create_tasks()¶

Returns a list of Lib.CreateTask.CreateTask. if no new task return a empty list.

get_delete_tasks()¶

Returns a list of Lib.DeleteTask.DeleteTask. if no new task return a empty list.

get_local_update_task(hour_start=None)¶

returns one Lib.LocalUpdateTask.LocalUpdateTask. not a list!
