TwitterAdForeman¶

class Lib.TwitterAdForeman.TwitterAdForeman(TaskQueue, TaskLock)¶

Bases: threading.Thread

Foreman extends threading.Thread. After starting, it will periodically update sessions & account settings from database, update TaskQueue, and update tasks.

TaskQueue = None¶

It is responsible for the communication between Lib.TwitterAdForeman.TwitterAdForeman thread and Lib.TwitterAdWorker.TwitterAdWorker threads. It is an object of Lib.MultiTubeQueue.MultiTubeQueue.

check()¶

Update :attr:TwitterSessions from local database by calling Lib.SessionHelper.update_sessions(). Update tasks and put into :attr:TaskQueue for Lib.TwitterAdWorker.TwitterAdWorker by calling Lib.TaskHelper.update_taskqueue().

run()¶

Implement the interface of Thread. it Periodically calls Lib.TwitterAdForeman.TwitterAdForeman.check().
