TwitterAdGenerator¶

This is the helper for generating campaigns for a given account.

Lib.TwitterAdGenerator.crtpd2refillcmp(account, generate_num)¶
Generate CreatePending campaigns for the given account.
It uses Lib.MyFunction.file2list() to parse three candidate files which is named after the username of the account.

For example: if you have a account with username ‘blar’. It will looking for the following files:

list_files/blar_user_list.txt

list_files/blar_intst_list.txt

list_files/blar_cntry_list.txt

list_files/common_user_list.txt

list_files/common_intst_list.txt

list_files/common_cntry_list.txt

With the xxxx_private_weight, the program will get a certain weight from private and common.

For example: your user_private_weight is 0.6. Then, the portion from each file is shown:

(assuming that you wanna 50 users)

blar_user_list.txt = 30 | common_user_list.txt = 20

It calls Lib.MyFunction.Username2Userid() to convert username to user id. It uses Lib.MyDict.INTEREST_ID_DICT to convert interest name to interest id. It uses Lib.MyDict.COUNTRY_ID_DICT to convert country name to country id.

param account:Specify the account.
param generate_num:
 Specify the number of campaigns to generate.
returns:The number of campaigns which are generated.
