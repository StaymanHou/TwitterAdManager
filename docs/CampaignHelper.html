CampaignHelper¶

This is the helper for Lib.TwitterCampaign.TwitterCampaign. Dealing with the operations related to TwitterCampaign.

Lib.CampaignHelper.create(campo)¶

Create a new campaign with an online campaign object.

..note:

The online campaign object is slitely diff. It's data attribute 
        only holds the value of a certain hour. Not a list.

        >>> campo.data = {'spend': 1.23,
        >>>                       'impressions': 1000,
        >>>                               'engagements': 123}
Lib.CampaignHelper.find_min_id(camp_list)¶

Return the min id among all ids of alive campaigns in this camp_list. Return 0 if there’s no camp in the list

Lib.CampaignHelper.generate_createpending_camp(account, generate_num)¶

Generate certain number of new campaigns and put into ‘createpending’. Calls Lib.TwitterAdGenerator.TwitterAdGenerator.crtpd2refillcmp().

Lib.CampaignHelper.get_create_payload(camp, twitter_session)¶

Return a create payload out of a campaign and a twitter_session for a twitter create request.

Lib.CampaignHelper.get_delete_payload(camp, twitter_session)¶

Return a delete payload out of a campaign and a twitter_session for a twitter delete request.

Lib.CampaignHelper.get_poor_performance_camp_list(account)¶

Return a list of all alive TwitterCampaigns of the account if poor_zscore_threshold <= -99. Otherwise, return the result of Lib.TwitterAdAnalyzer.TwitterAdAnalyzer.GetPoorPfmcCmpListIMPBased() call.

Lib.CampaignHelper.kill(camp)¶

Kill the given campaign. Set it into dead, active = False, and end_time = now.

Lib.CampaignHelper.set_campaign_create_fail(camp)¶

Set a camp into createfail

Lib.CampaignHelper.set_campaign_created(camp, new_id)¶

Set a camp into alive

Lib.CampaignHelper.set_campaign_delete_fail(camp)¶

Set a camp into deletefail

Lib.CampaignHelper.set_campaign_deleted(camp)¶

Set a camp into dead

Lib.CampaignHelper.set_delete_pending(camp_list)¶

Set the campaigns in the camp_list into ‘deletepending’.

Lib.CampaignHelper.update(campl, campo)¶

Update the data of an local campaign with an online campaign object.

..note:

The online campaign object is slitely diff. It's data attribute 
        only holds the value of a certain hour. Not a list.

        >>> campo.data = {'spend': 1.23,
        >>>                       'impressions': 1000,
        >>>                               'engagements': 123}
